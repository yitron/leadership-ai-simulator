# Leadership AI Scenario Simulator

AI-powered chatbot/scenario simulator for a 1-day course on **Leading AI-Enabled Teams**, built using the Handshaw Instructional Design Model.

## Overview

Targeted at adult educators and corporate trainers, this prototype delivers 3–5 corporate scenarios with AI-generated, contextual feedback to help leaders manage AI-enabled teams effectively.

## Tech Stack

- React / HTML chatbot prototype
- Anthropic API for AI-generated feedback
